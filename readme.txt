Activate this to show the theme customizer in block themes.

== Description ==

This plugin automatically shows the theme customizer after it was removed in WordPress 5.9.

== Frequently Asked Questions ==

= I don't want the customizer anymore, how do I remove it? =

Deactivate this plugin.

= Does this plugin store any data? =

No.
